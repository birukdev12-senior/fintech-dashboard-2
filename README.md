## ✨ Features
- 📊 **Interactive Charts** — Line, Bar & Doughnut charts via Chart.js
- 📈 **KPI Cards** — Real-time stat cards with trend indicators (▲/▼)
- 🧭 **Collapsible Sidebar** — Responsive navigation with icon support
- 📋 **Recent Transactions** — Sortable table with status badges
- ♿ **Fully Accessible** — ARIA labels, skip‑to‑content link, keyboard navigation
- 🌙 **Responsive Design** — Works on mobile, tablet & desktop
